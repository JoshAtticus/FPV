# FPV
FPV is a camera app for the Quest 3 (and possibly the Quest 3s, but this is untested! Please open an issue if you've tested it on a Quest 3s). 

## Features
- Photo & Video Capture
- Spatial Photo Capture
- Avatar Capture
- Lens Selector
- Settings page for resolution, bitrate, audio quality, app settings etc
